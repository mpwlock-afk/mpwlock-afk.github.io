## Per concurrent (kort)

- **Proto (protohologram.com)** — de SEO-benchmark. Sterke resources/blog, named case studies (DKNY, artists), enorme earned-media/PR-motor (celebrity "beam-ins" → backlinks), e-commerce-achtige productpagina's met specs.
- **Hypervsn (hypervsn.com)** — industrie-gesegmenteerde solution-pagina's (1 URL per verticaal = keyword-dekking), resource center met gated assets, case-study library, reseller/partner-netwerk (linknetwerk).
- **ARHT (arht.tech)** — public-company persritme = constante stroom indexeerbare, link-aantrekkende news; partner-venue-netwerk; enterprise/event cases.
- **Looking Glass / Realfiction(Dreamoc)** — secundair; Looking Glass sterk op developer-docs/long-tail.

## Top-aanbevelingen (overlap met ACTION-PLAN §3)

1. `/sectors` splitsen in losse URL's — ✅ gedaan.
2. `/cases` → echte case-hub met `/cases/[slug]` (named brands = #1 autoriteits/backlink-asset, nu onbenut).
3. Resources/Insights-hub lanceren (informational long-tail; nu 0).
4. Press/News-room (ING-opening, BMW, government deployments als nieuws → pers pitchen).
5. Structured data sitebreed — ✅ Org/Product/Breadcrumb gedaan; FAQ/Article nog.
6. On-page fundament — ✅ canonical/sitemap/OG/meta gedaan.
7. Comparison + bottom-funnel (vs Proto/Hypervsn, pricing/quote).
8. Video/demo-gallery + glossary + interne hub-and-spoke linking.

**Kerninzicht:** concurrenten verdelen content over tientallen indexeerbare URL's; Holoconnects propte alles in een handvol pagina's. Die hub-and-spoke-architectuur repliceren (sectors ✅ → cases → blog → comparisons) is de grootste organische hefboom.
